## Data loading considerations

This set of topics provides best practices, general guidelines, and important considerations for bulk data loading using the [COPY INTO <table>](https://docs.snowflake.com/sql-reference/sql/copy-into-table) command. It is intended to help simplify and optimize importing of data from data files into Snowflake tables.

> [!note] Note
> Most considerations in this set of topics pertain to bulk data loading only. However, the [Preparing your data files](https://docs.snowflake.com/user-guide/data-load-considerations-prepare) topic applies to both bulk loading and continuous loading using [Snowpipe](https://docs.snowflake.com/user-guide/data-load-snowpipe-intro).
